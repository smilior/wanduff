⛳ ワンダフの使い方

ラウンド中のミスを詳細に記録。独自のデータ分析で、あなたのスコアアップを最短距離でサポートします。

01

play_circle ラウンドを開始する

ホーム画面の「ラウンドを始める」をタップ。プレー日・コース名・同伴者などを入力してスタート。

done
02

edit_square ホール毎にスコアを記録

各ホール終了後、打数・パット数・フェアウェイキープを入力。ミスが起きたシーンも選択。

done
03

error_outline ミスシーンを選ぶ

OB・池・バンカー・3パットなどシーン別にタップ。後からミス傾向が一目でわかる.

done
04

assignment_turned_in ラウンド終了・振り返り

18ホール後に「終了」をタップ。スコアカードとミス集計が自動生成され、課題が視覚化。

done
05

analytics ミスを振り返る

ホームの「ミスを振り返る」から過去のデータを確認。頻出ミスを把握して練習に活かす。

done
